# Pulsar-V in-order core - Crab

Welcome to official **Crab Core** documentation, a in-order core implementing RV64GC.

## About Project

Crab is a in-order Risc-V core (hart) desing to implement RV64GC ISA for GP CPU usage. It is performance optimized both for FPGA and ASIC implementation.
